# Homebrew Tap for clickup-cli

CLI for the ClickUp API, optimized for AI agents.

## Install

```bash
brew tap nicholasbester/clickup-cli
brew install clickup-cli
```

## Usage

```bash
clickup setup --token pk_your_token
clickup auth whoami
clickup task list --list 12345
```

See [clickup-cli](https://github.com/nicholasbester/clickup-cli) for full documentation.
